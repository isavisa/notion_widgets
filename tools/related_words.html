Related Words Tool

    In this example, the tool prompts the user to enter a word, and when they click the "Fetch Related Words" button, it fetches a list of synonyms (related words) for that input word using the Datamuse API. The related words are then displayed in a list below the input field. Keep in mind that this is a basic example, and you might want to add error handling and improve the user interface. Additionally, you could explore other APIs or sources for more comprehensive word-related data.
